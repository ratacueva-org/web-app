# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by emailing christian.serrano.puertos@gmail.com.

We will acknowledge receipt within 48 hours and provide a detailed response within 5 business days.

Please do not disclose security vulnerabilities publicly until they have been addressed by the maintainers.

## Responsible Disclosure

We kindly request that you:
- Provide sufficient details to allow us to reproduce and fix the issue
- Give us reasonable time to respond before making any information public
- Make a good faith effort to avoid privacy violations, data destruction, and service disruptions
